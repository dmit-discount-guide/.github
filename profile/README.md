## 先把坑踩在前头：买 DMIT 常见的几个冤枉钱

在说省钱之前，先说几个让人多花钱的误区：

**误区一：以为优惠码可以叠加**
不行。DMIT 系统每次下单只能用一个优惠码。建议提前比较几个，选折扣最大那个。

**误区二：月付+优惠码，后来想改年付**
这个坑挺深的——如果你用优惠码购买了月付套餐，后来想改成年付，优惠折扣会失效，恢复原价。**所以如果打算用码，建议一开始就选年付或季付**。

**误区三：只看价格，不看线路**
DMIT 旗下有 Premium（CN2 GIA）、Eyeball（CMIN2/9929）、Tier 1（国际线路）三条线路体系，价格差距可以很大，但适用场景完全不同。买错了线路，就算买到再便宜的套餐也是白给。

**误区四：错过补货时间**
DMIT 的特价限量套餐（比如 LAX.Pro.WEE 年付 $36.9 这类）不是常备的，属于限量放货，出了就没了，等到缺货再来找就晚了。
